# netease-cloud-music

网抑云音乐 Golang API 接口

# 重要声明

本项目仅供个人学习使用.

- [x] 支持接口参数加解密
- [x] “音乐合伙人”自动测评

- [ ] 刷歌
- [ ] 音乐下载
- [ ] vip每日签到
- [ ] vip日常任务完成(待考虑)
